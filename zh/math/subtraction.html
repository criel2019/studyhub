1. 减法的意义

减法是加法的逆运算，从一个数中去掉一部分，求剩余部分的运算。

  • 被减数 - 减数 = 差
  • 例：9 - 4 = 5（9是被减数，4是减数，5是差）
  • 减法可以表示"减少"、"剩余"、"相差多少"等实际问题

2. 减法性质

  • 一个数减去两个数的和：a - (b + c) = a - b - c
    例：100 - (23 + 37) = 100 - 23 - 37 = 40
  • 连续减去两个数可以换序：a - b - c = a - c - b
    例：200 - 76 - 24 = 200 - 24 - 76 = 176 - 76 = 100

3. 退位减法

当某数位上的被减数 < 减数时，需要向高位借1，这就是退位。借来的1在本位等于10。

例题：计算 603 - 278
  6 0 3
- 2 7 8
───────
  3 2 5

步骤：
个位：3 < 8，向十位借1，但十位是0，继续向百位借1。百位6变5，十位0变9，再借给个位：13 - 8 = 5
十位：9 - 1（已借出）- 7 = 1 → 实际9 - 7 - 1 = 1 → 借后：(9-1) - 7 = 2
百位：5 - 2 = 3

答：603 - 278 = 325

4. 减法的验算

  • 方法：差 + 减数 = 被减数（用加法检验减法）
  • 例：325 + 278 = 603 ✓

5. 练习题

  1. 计算：802 - 465
  2. 利用减法性质简算：356 - 98 - 2
  3. 学校图书馆有书480本，借出163本，还剩多少本？
答案
  1. 802 - 465 = 337
  2. 356 - (98 + 2) = 356 - 100 = 256
  3. 480 - 163 = 317（本）
考前5分钟检查清单
  • 退位时从上一位借1当10
  • 验算：差+减数=被减数
  • 减法不满足交换律（a−b≠b−a）
  • 两数相等则差为0
