1. 小数的意义与读法

  • 小数是分母为10、100、1000……的分数的另一种表示形式
  • 小数点左边是整数部分，右边依次是十分位、百分位、千分位……
  • 读法：整数部分按整数读，小数点读"点"，小数部分逐个数字读出
小数读法分数形式
0.3零点三3/10
1.25一点二五125/100 = 5/4
0.007零点零零七7/1000

2. 小数的四则运算

  • 加减法：小数点对齐，再按整数方法计算
    例：3.45 + 1.6 → 对齐 → 3.45 + 1.60 = 5.05
  • 乘法：先按整数乘，再看两个因数共有几位小数，积就有几位小数
    例：1.2 × 0.3：12 × 3 = 36，共2位小数 → 0.36
  • 除法：将除数化为整数（同时扩大被除数相同倍数）
    例：4.8 ÷ 0.6 = 48 ÷ 6 = 8

3. 小数与分数互化

  • 小数 → 分数：去掉小数点，以10的幂为分母，再化简
    例：0.75 = 75/100 = 3/4
  • 分数 → 小数：分子除以分母
    例：3/8 = 3 ÷ 8 = 0.375
  • 循环小数：1/3 = 0.333… = 0.3̄（不能化为有限小数）

4. 练习题

  1. 计算：12.36 + 5.8
  2. 计算：2.5 × 0.04
  3. 将 0.625 化为最简分数
  4. 将 7/20 化为小数
答案
  1. 12.36 + 5.8 = 18.16
  2. 2.5 × 0.04 = 0.1
  3. 0.625 = 625/1000 = 5/8
  4. 7 ÷ 20 = 0.35
