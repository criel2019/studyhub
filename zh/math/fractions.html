½ 分数完全讲解

📚 小学～初中数学 🎯 重要概念：分数类型、约分通分、四则运算

1. 分数的基本概念

  • 真分数：分子 < 分母（如：3/5，值小于1）
  • 假分数：分子 ≥ 分母（如：7/4，值大于等于1）
  • 带分数：整数部分 + 真分数（如：1¾）
  • 假分数与带分数可以互化：7/4 = 1¾

2. 约分与通分

  • 约分：分子分母同除以最大公因数（GCD），化为最简分数
    例：12/18 ÷ 6/6 = 2/3
  • 通分：将两个分数化为同分母分数（使用最小公倍数LCM）
    例：1/4 和 1/6：LCM(4,6) = 12，所以 3/12 和 2/12

3. 分数的四则运算

  • 加减法：先通分，再加减分子，分母不变
    例：1/3 + 1/4 = 4/12 + 3/12 = 7/12
  • 乘法：分子×分子，分母×分母，再化简
    例：2/3 × 3/4 = 6/12 = 1/2
  • 除法：乘以除数的倒数
    例：3/4 ÷ 2/5 = 3/4 × 5/2 = 15/8
例题：计算 2½ + 1¾

化为假分数：5/2 + 7/4

通分：10/4 + 7/4 = 17/4

化为带分数：17/4 = 4¼

答：2½ + 1¾ = 4¼

4. 练习题

  1. 化简：24/36
  2. 计算：3/5 + 2/3
  3. 计算：5/6 × 3/10
  4. 计算：7/8 ÷ 7/4
答案
  1. 24/36 = 2/3
  2. 3/5 + 2/3 = 9/15 + 10/15 = 19/15 = 1 4/15
  3. 5/6 × 3/10 = 15/60 = 1/4
  4. 7/8 ÷ 7/4 = 7/8 × 4/7 = 28/56 = 1/2
