🌱 为什么要学化学？
直觉理解

化学是原子重新排列的游戏。化学反应中原子不会凭空产生或消失，只是结合方式改变(质量守恒定律)。看化学式时先问：哪些原子各有几个是怎么结合的？

化学是解释物质是什么、如何变化的科学。药物如何治病？铁为何生锈而金不生锈？电池如何储电？这些都有化学解答。酸碱概念与血液pH调节直接相关，氧化还原是电池和燃料电池的工作原理。

⚡ 30秒核心摘要
  • 物质分类：纯净物（单质·化合物）vs 混合物（均匀·非均匀）
  • 化学键：离子键（金属+非金属，电子转移），共价键（非金属+非金属，电子共用）
  • 化学方程式：反应物 → 生成物。原子数守恒（配平系数）
  • 酸碱：酸（释放H⁺），碱（释放OH⁻）。中和：酸+碱→水+盐
  • 氧化还原：氧化（失去电子），还原（获得电子）
🏛 概念的诞生

拉瓦锡（1789年）通过密封燃烧实验精确称量，证明了质量守恒定律，推翻了燃素说——化学史上第一次范式革命。近一个世纪后，门捷列夫于1869年发表元素周期表，特意为尚未发现的元素留下空格。镓（1875年）和锗（1886年）按其预测被发现时，元素周期表成为科学史上最伟大的预测性成就之一。

1. 原子结构

电子-1≈0核外电子层
  • 原子序数(Z)：等于质子数，决定元素种类
  • 质量数(A)：质子数 + 中子数
  • 同位素：质子数相同，中子数不同的原子（如 ¹²C 和 ¹⁴C）
  • 离子：得失电子后带电的原子（阳离子失电子，阴离子得电子）

2. 元素周期律

元素按原子序数排列，性质相似的元素周期性出现。

  • 第ⅠA族（碱金属）：Li, Na, K…　活泼金属，最外层1个电子
  • 第ⅦA族（卤族）：F, Cl, Br…　活泼非金属，最外层7个电子
  • 第0族（稀有气体）：He, Ne, Ar…　化学性质极不活泼
  • 同周期从左到右：金属性减弱，非金属性增强
  • 同族从上到下：金属性增强，非金属性减弱

3. 化学键

  • 离子键：阳离子与阴离子间的静电引力。金属+非金属。例：NaCl
  • 共价键：原子间共用电子对。非金属间。例：H₂O、CO₂
  • 极性键：电负性不同的原子间，电子对偏向电负性大的原子
  • 金属键：金属离子与自由电子间的作用力，导电导热性的来源

4. 化学反应类型

  • 化合反应：A + B → AB（多变一）
    例：2H₂ + O₂ → 2H₂O
  • 分解反应：AB → A + B（一变多）
    例：2H₂O →(电解) 2H₂↑ + O₂↑
  • 置换反应：单质 + 化合物 → 新单质 + 新化合物
    例：Fe + CuSO₄ → FeSO₄ + Cu
  • 复分解反应：AB + CD → AD + CB
    例：NaCl + AgNO₃ → AgCl↓ + NaNO₃

化学方程式配平 高考

例：配平 Fe + O₂ → Fe₂O₃

分析：铁原子：左1，右2；氧原子：左2，右3

设系数：4Fe + 3O₂ → 2Fe₂O₃

验证：铁 4=4✓　氧 6=6✓

答：4Fe + 3O₂ → 2Fe₂O₃

5. 酸碱盐与pH 高考

  • 酸：溶液中含有H⁺的物质。pH < 7。例：盐酸（HCl）、硫酸（H₂SO₄）
  • 碱：溶液中含有OH⁻的物质。pH > 7。例：NaOH、Ca(OH)₂
  • 中性：pH = 7（纯水）
\[ \text{pH} = -\lg[\text{H}^+] \]

中和反应：酸 + 碱 → 盐 + 水
例：HCl + NaOH → NaCl + H₂O

❌ 常见错误 — 氧化剂与还原剂混淆
氧化剂本身被氧化（因为它"使别人氧化"） 氧化剂本身被还原（得到电子），同时使另一种物质被氧化
名称描述的是它对别人的作用，不是对自己。口诀：氧化失去电子，还原得到电子。氧化剂得电子→自身被还原。

6. 练习题

  1. 原子序数17、质量数35的氯原子：质子数、中子数、电子数各是多少？
  2. Na（Z=11）的电子排布是怎样的？
  3. 配平：Al + HCl → AlCl₃ + H₂↑
  4. [H⁺] = 0.001 mol/L 时，pH是多少？
答案
  1. 质子17，中子18，电子17
  2. 2、8、1（K层2个，L层8个，M层1个）
  3. 2Al + 6HCl → 2AlCl₃ + 3H₂↑
  4. pH = -lg(0.001) = -lg(10⁻³) = 3（酸性）
🔗 通往下一个概念的桥梁

原子和分子结合形成的有机物，如何变成有生命的细胞？

蛋白质、DNA、脂质都是以碳为骨架的有机化合物。共价键和离子键的原理，正是生命分子机器的基础。

化学是生命科学、医药与材料工程的共同语言。掌握元素周期表和化学方程式之后，细胞层面的化学反应——生物化学——将触手可及。

考前5分钟检查清单
  • 化学方程式：两边原子数必须相等
  • 1 mol = 6.02×10²³ 个粒子（阿伏伽德罗常数）
  • pH：7=中性 · <7=酸性 · >7=碱性
  • 氧化还原同时发生（得失电子）
