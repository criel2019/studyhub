⚡ 物理完全讲解

📚 初中・高中物理 🎯 重要概念：力、运动、能量、波动、电学
常见误解

误解： 没有力，物体就会停止运动。

事实： 伽利略和牛顿发现：没有力，物体保持当前状态。地球上物体停下来是因为摩擦力(一种力)在起作用。

1. 力（Force）

力是改变物体运动状态或使物体发生形变的原因。力是矢量，有大小和方向。

  • 重力：\(G = mg\)（g = 9.8 m/s²，竖直向下）
  • 弹力：弹簧的弹力 \(F = kx\)（胡克定律）
  • 摩擦力：\(f = \mu N\)（μ为动摩擦因数，N为支持力）
  • 力的单位：牛顿（N），1 N = 1 kg·m/s²

2. 牛顿运动定律 高考

第一定律（惯性定律）

一切物体在没有受到外力或合外力为零时，总保持静止状态或匀速直线运动状态。
  • 惯性：物体保持原有运动状态的性质，质量越大惯性越大
  • 例：急刹车时乘客向前倾（惯性）

第二定律（加速度定律）

\[ F = ma \]

F：合外力(N)，m：质量(kg)，a：加速度(m/s²)

例题：质量4 kg的物体受12 N合力，加速度为多少？

\(a = F/m = 12/4 = 3\) m/s²

第三定律（作用力与反作用力）

两物体之间的作用力和反作用力总是大小相等、方向相反、作用在同一条直线上，分别作用在两个物体上。

3. 匀变速运动公式 高考

\[ v = v_0 + at \] \[ s = v_0 t + \frac{1}{2}at^2 \] \[ v^2 = v_0^2 + 2as \]

自由落体：初速度为零，加速度 g = 9.8 m/s²（忽略空气阻力）

下落高度：\(h = \dfrac{1}{2}gt^2\)

4. 能量

动能：\(E_k = \dfrac{1}{2}mv^2\)

重力势能：\(E_p = mgh\)
机械能守恒定律：在只有重力或弹力做功时，动能与势能之和保持不变。 \[E_k + E_p = \text{常数}\]

5. 波动

\[ v = f\lambda \]

v：波速(m/s)，f：频率(Hz)，λ：波长(m)

  • 横波：振动方向⊥传播方向（光、水波）
  • 纵波：振动方向∥传播方向（声波）
  • 声音在空气中的速度约为340 m/s（15°C）
  • 光速：真空中 \(c = 3 \times 10^8\) m/s

6. 电学

欧姆定律

\[ U = IR \]

U：电压(V)，I：电流(A)，R：电阻(Ω)

串联：\(R_{总} = R_1 + R_2 + \cdots\)
并联：\(\dfrac{1}{R_{总}} = \dfrac{1}{R_1} + \dfrac{1}{R_2} + \cdots\)
电功率：\(P = UI = I^2R = \dfrac{U^2}{R}\)

7. 练习题

  1. 质量2 kg的物体受合力10 N，求加速度
  2. 初速度0，加速度5 m/s²，求4 s末速度和位移
  3. 电压9 V，电阻3 Ω，求电流和功率
答案
  1. \(a = 10/2 = 5\) m/s²
  2. \(v = 0 + 5 \times 4 = 20\) m/s；\(s = \frac{1}{2} \times 5 \times 16 = 40\) m
  3. \(I = 9/3 = 3\) A；\(P = 9 \times 3 = 27\) W
考前5分钟检查清单
  • 牛顿第二定律：F=ma，单位 N=kg·m/s²
  • 匀变速运动4公式：先确定未知量再选公式
  • 机械能守恒：无摩擦时 KE+PE=恒定
  • 欧姆定律 V=IR · 串联（R相加）vs 并联（1/R相加）
