🧬 生物完全讲解

📚 初中・高中生物 🎯 重要概念：细胞、光合作用、遗传、进化

1. 细胞生物学

细胞学说

  1. 所有生物都由一个或多个细胞组成
  2. 细胞是生命活动的基本单位
  3. 所有细胞来自已有细胞的分裂

原核细胞与真核细胞

特征原核细胞真核细胞
细胞核无核膜，有拟核有核膜包被的细胞核
大小小（1〜10 μm）大（10〜100 μm）
细胞器少，无膜结构多种有膜细胞器
代表生物细菌、蓝藻动物、植物、真菌

主要细胞器的功能

  • 细胞核：含有DNA，是遗传信息中心，控制细胞生命活动
  • 线粒体：有氧呼吸的主要场所，合成ATP，"细胞的能量工厂"
  • 核糖体：蛋白质合成的场所
  • 内质网：粗面内质网（蛋白质加工）、滑面内质网（脂质合成）
  • 高尔基体：蛋白质的加工、分拣和运输，"细胞的邮局"
  • 叶绿体：（植物细胞）光合作用的场所，含叶绿素
  • 细胞膜：磷脂双分子层，具有选择透过性，控制物质进出
  • 细胞壁：（植物细胞）主要成分纤维素，起支持保护作用
  • 液泡：（植物细胞）储存物质，中央大液泡维持细胞膨压

2. 光合作用与细胞呼吸

光合作用

6CO₂ ＋ 6H₂O ＋ 光能 → C₆H₁₂O₆ ＋ 6O₂
  • 光反应（基粒类囊体）：吸收光能，分解水，产生ATP和NADPH，释放O₂
  • 暗反应/卡尔文循环（叶绿体基质）：利用ATP和NADPH固定CO₂，合成有机物（葡萄糖）
  • 叶绿素吸收红光和蓝紫光，反射绿光（植物呈绿色的原因）

细胞呼吸（有氧呼吸）

C₆H₁₂O₆ ＋ 6O₂ → 6CO₂ ＋ 6H₂O ＋ ATP（约36〜38个）
  • 糖酵解（细胞质基质）：葡萄糖 → 2丙酮酸，净产生2ATP
  • 柠檬酸循环（线粒体基质）：丙酮酸 → CO₂，产生NADH、FADH₂
  • 氧化磷酸化（内膜）：NADH、FADH₂提供电子，产生约34ATP，O₂是最终电子受体

3. DNA、基因与蛋白质合成

  • DNA：脱氧核苷酸，双螺旋结构，碱基：A（腺嘌呤）、T（胸腺嘧啶）、G（鸟嘌呤）、C（胞嘧啶）
  • 碱基互补配对：A-T，G-C（DNA）；A-U，G-C（RNA）
  • 基因：携带遗传信息的DNA片段
  • 转录：DNA → mRNA（在细胞核中进行）
  • 翻译：mRNA → 蛋白质（在核糖体上进行）。密码子（3个碱基）决定氨基酸

4. 孟德尔遗传定律

基本术语：
  • 显性基因(A)：只需一个拷贝即可表现性状
  • 隐性基因(a)：两个拷贝(aa)时才表现性状
  • 基因型：基因组合（AA、Aa、aa）
  • 表现型：实际表现的性状
  • 纯合子：相同等位基因（AA 或 aa）
  • 杂合子：不同等位基因（Aa）

分离定律（孟德尔第一定律）

Aa × Aa 的杂交
   A   a
A|AA|Aa|
a|Aa|aa|
基因型比：AA:Aa:aa = 1:2:1
表现型比：显性:隐性 = 3:1

5. 进化与自然选择

达尔文自然选择学说的四个要素：
  1. 个体间存在变异
  2. 变异是可以遗传的
  3. 生物的繁殖能力超过环境承载量，存在生存竞争
  4. 有利变异的个体存活和繁殖机会更多（适者生存）
进化的证据：
  • 化石记录：显示生物形态随时间变化
  • 同源器官：外形不同但有共同祖先（人手、海豚鳍、蝙蝠翼）
  • 分子生物学证据：DNA相似性反映进化关系
生物学习要点： 光合作用和细胞呼吸的方程式要精确记忆，注意两者是互逆的关系。遗传题要画棋盘格（遗传图解）来避免计算错误。细胞器要名称和功能配对记忆。
