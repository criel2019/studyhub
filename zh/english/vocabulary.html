📝 英语词汇学习

🌐 初中·高中英语 🎯 核心策略：词根词缀·语境推断·搭配·词汇扩展

1. 词汇能力是什么

词汇能力不只是"认识多少单词"，而是在语境中准确理解词义并灵活运用的能力。
  • 被动词汇：读或听到时能理解意思的词（阅读理解的关键）
  • 主动词汇：能在写作和口语中主动使用的词（英语写作的核心）
  • 高考标准：约需掌握3,500个以上词汇

2. 词根、词缀与构词法

了解词缀规律，能推测陌生单词的意思：
  • 前缀：un-(not: unhappy), dis-(not: disagree), re-(again: review), pre-(before: predict), over-(too much: overwork), mis-(wrong: mislead)
  • 后缀（名词）：-tion/-sion(action), -ment(agreement), -ness(happiness), -ity(ability)
  • 后缀（形容词）：-ful(useful), -less(careless), -able(possible), -ous(famous)
  • 后缀（动词）：-ize(realize), -ify(simplify), -en(widen)
派生词组示例：
create → creation(名词) → creative(形容词) → creatively(副词) → creativity(名词)
depend → dependent(形容词) → independence(名词) → independently(副词)

3. 语境推断词义

遇到陌生单词时，利用上下文推断意思的策略：
  • 定义型："X, which means / that is / in other words" → 直接找定义
  • 举例型："such as, for example, including" → 从例子推断上位概念
  • 对比型："but, however, although, unlike" → 从反义推断
  • 因果型："because, therefore, as a result" → 通过因果关系缩小意思范围
练习："The new policy was contentious; many opposed it while others strongly supported it."
→ opposed vs supported → 意见相反 → contentious = 有争议的，引起争论的

4. 高频词汇主题分类

  • 心理·情感：anxiety(焦虑), reluctant(不情愿), contempt(轻蔑), empathy(共情), nostalgic(怀旧的)
  • 社会·文化：diverse(多样的), prevailing(占主导的), conform(顺从), prejudice(偏见), hierarchy(等级制度)
  • 科学·技术：phenomenon(现象), hypothesis(假设), inevitable(不可避免的), fluctuate(波动)
  • 逻辑·连接词：nevertheless(尽管如此), whereas(然而), consequently(因此), simultaneously(同时)

5. 多义词与搭配（Collocation）

  • 多义词：根据语境意思不同的词
    "light" → 光/轻的/明亮的/点燃，"run" → 跑/经营/流动
  • 搭配：经常一起使用的词组合
    make a decision(✓) vs do a decision(✗)
    take a risk, pay attention, give advice, have a discussion
  • 近义词辨析："big/large/great"——注意使用场合和语义差异

6. 词汇扩展策略

  • 结合例句和派生词记录：在词汇本中同时写例句和派生词
  • 主题分组记忆：把相关词汇归类学习（如：环境词汇组）
  • 广泛阅读：日常接触英语课文、试卷、英文报刊
  • 实践运用：在英语写作中主动使用新词，加深记忆
💡 英语词汇学习要点：词汇最好在语境中记忆，死背单词表效果有限。阅读时遇到陌生词，先试着从上下文推断，再查词典确认。注意单词的搭配（如 make a decision 而非 do a decision），这样才能真正活学活用。
