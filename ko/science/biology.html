연습문제

★☆☆ 기초 Q. ATP에서 에너지가 방출될 때 일어나는 반응을 화학식으로 쓰시오.

정답 확인
ATP → ADP + Pi (무기인산). 인산 결합이 끊어지며 에너지 방출. 역반응(ADP + Pi → ATP)으로 에너지 저장

★★☆ 심화 Q. 광합성의 원료 2가지와 생성물 2가지를 쓰시오.

정답 확인
원료: 물(H₂O) + 이산화탄소(CO₂). 생성물: 포도당(C₆H₁₂O₆) + 산소(O₂). 빛에너지 → 화학에너지 변환

★★★ 도전 Q. DNA 반보존적 복제의 의미를 설명하고, 왜 반보존적인지 이유를 쓰시오.

정답 확인
원래 DNA 2가닥이 각각 분리→각각 새 가닥의 주형이 됨. 딸세포 DNA = 원래 가닥 1개 + 새 가닥 1개. 반(半)보존 = 절반은 원래 가닥 보존
생명 활동 핵심 암기

ATP = 아데노신 3인산. 광합성 공식: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂. 세포 호흡(역반응): 포도당 분해 → ATP 생성. DNA 이중나선: A-T 쌍, G-C 쌍. 반보존적 복제: 가닥 분리 후 각각 주형.
