탄생 배경

화학은 연금술사들의 금 제조 시도에서 탄생했습니다. 라부아지에가 1789년 산소를 발견하고 질량 보존 법칙을 제시하며 연금술을 과학으로 전환시켰고, '현대 화학의 아버지'로 불리게 되었습니다.

🌱 왜 화학을 배우는가?

화학은 물질이 무엇이고 어떻게 변하는지 설명하는 과학입니다. 약이 어떻게 병을 치료하는지, 왜 철은 녹슬고 금은 녹슬지 않는지, 배터리가 어떻게 전기를 저장하는지 — 모두 화학의 언어로 설명됩니다. 산·염기 개념은 혈액의 pH 조절에 직결되고, 산화·환원은 연료전지·배터리 작동 원리입니다. 화학을 알면 물질 세계를 원자 수준에서 이해하는 눈이 생깁니다.

💡 이걸로 이해하면 됩니다
화학 결합 = 짝꿍 찾기

원소들은 최외각 전자를 채우려고 한다 (8개가 완성). 이온 결합 = 전자를 완전히 주거나 받는 것(Na+ + Cl− → 소금). 공유 결합 = 전자를 사이좋게 공유(H₂O, CO₂). 혼자서 8개가 안 되니까 짝꿍을 찾아 완성하는 것이다.

⚡ 핵심 30초 요약
  • 물질 분류: 순물질(원소·화합물) vs 혼합물(균일·불균일)
  • 화학 결합: 이온결합(금속+비금속), 공유결합(비금속+비금속)
  • 화학반응식: 반응물 → 생성물. 원자수 보존 (계수 맞추기)
  • 산·염기: 산(H⁺ 내놓음), 염기(OH⁻ 내놓음). 중화: 산+염기→물+염
  • 산화·환원: 산화(전자 잃음·산소 얻음), 환원(전자 얻음·산소 잃음)
📊 수능 출제 포인트
  • 출제 빈도: 화학Ⅰ 선택 시 매년 7~9문항 (전체 20문항 중 최다)
  • 핵심 유형: 원소 주기성(이온화 에너지·전기음성도), 화학 결합 비교, 산화수 계산, 중화 적정
  • 고난도 포인트: 루이스 전자 구조식과 결합 모형 — 4점 문항에 자주 등장
  • 실수 주의: 산화·환원에서 "산소를 잃으면 환원" 공식 암기만 하고 원리를 이해 못할 경우 변형 문제에서 오답
🏛️ 개념의 탄생

현대 화학의 기틀을 세운 것은 프랑스의 라부아지에(Lavoisier, 1743~1794)입니다. 그는 연소가 "플로지스톤(가상의 불 원소)" 때문이 아니라 산소와의 결합임을 실험으로 증명하고, "질량 보존의 법칙"을 정립했습니다. 러시아의 멘델레예프(Mendeleev, 1869)는 당시 알려진 63개 원소의 패턴을 발견해 주기율표를 만들었는데, 빈칸으로 두었던 자리의 원소(갈륨, 게르마늄)가 실제로 발견되면서 주기율표의 신뢰성이 입증됐습니다.

학습자 질문

화학 반응에서 에너지가 방출된다는 게 정확히 무슨 뜻인가요?

결합이 형성될 때는 에너지가 방출되고, 끊어질 때는 에너지가 필요합니다. 반응 전 결합들의 에너지 합보다 반응 후 결합들의 에너지 합이 작으면 그 차이만큼 에너지가 방출됩니다. 이것이 발열 반응의 원리입니다.

1. 물질의 분류 [10통과01-01]

순물질(Pure Substance): 한 가지 성분으로만 이루어진 물질
  • 원소(Element): 더 이상 다른 물질로 분해되지 않는 순물질
    예: 수소(H₂), 산소(O₂), 철(Fe), 금(Au), 탄소(C)
  • 화합물(Compound): 두 가지 이상의 원소가 화학적으로 결합한 순물질
    예: 물(H₂O), 이산화탄소(CO₂), 소금(NaCl), 설탕(C₁₂H₂₂O₁₁)

혼합물(Mixture): 두 가지 이상의 순물질이 화학 반응 없이 섞인 것
  • 균일 혼합물(Homogeneous): 성분이 고르게 섞임. 예: 소금물, 공기, 합금
  • 불균일 혼합물(Heterogeneous): 성분이 고르지 않게 섞임. 예: 흙탕물, 우유, 화강암

2. 원자 구조 [10통과01-02]

원자는 물질을 이루는 기본 입자로, 더 이상 화학적으로 쪼갤 수 없는 단위입니다.

원자의 구성
  • 원자핵: 원자 중심에 위치. 매우 작지만 원자 질량의 대부분을 차지.
    • 양성자(Proton): +전하, 원자번호 결정
    • 중성자(Neutron): 전하 없음, 질량수에 기여
  • 전자(Electron): −전하, 원자핵 주위의 궤도를 돌며 화학 결합에 참여
핵심 관계식
원자번호(Z) = 양성자 수 = 전자 수 (중성 원자의 경우)
질량수(A) = 양성자 수 + 중성자 수
중성자 수 = 질량수 − 원자번호
예제: 탄소(C)의 원자번호는 6, 질량수는 12. 중성자 수는?
중성자 수 = 12 − 6 = 6개

동위원소(Isotope)

같은 원자번호(양성자 수 동일)를 가지지만 질량수(중성자 수)가 다른 원자
  • 수소의 동위원소: ¹H(수소, 중성자 0), ²H(중수소, 중성자 1), ³H(삼중수소, 중성자 2)
  • 탄소의 동위원소: ¹²C(중성자 6), ¹³C(중성자 7), ¹⁴C(중성자 8, 방사성)
  • 동위원소는 화학적 성질이 같고 물리적 성질(질량)이 다름

3. 주기율표 핵심 [12화학I01-01]

주기율표는 원소를 원자번호 순서로 배열하여 원소의 성질이 주기적으로 반복됨을 보여줍니다.

주기(Period)와 족(Group)
  • 주기: 가로줄 (1~7주기). 같은 주기의 원소는 전자 껍질 수가 같음
  • 족: 세로줄 (1~18족). 같은 족의 원소는 최외각 전자 수가 같아 화학적 성질이 유사함

주요 족의 특성
  • 1족 — 알칼리 금속: Li, Na, K 등. 반응성이 매우 크고, 물과 격렬히 반응하여 H₂ 발생
  • 2족 — 알칼리 토금속: Be, Mg, Ca 등. 반응성이 크나 1족보다는 낮음
  • 17족 — 할로젠: F, Cl, Br, I 등. 비금속, 최외각 전자 7개, 반응성 큼
  • 18족 — 비활성 기체(Noble Gas): He, Ne, Ar 등. 최외각 전자 8개(He는 2개), 화학 반응 거의 안 함

4. 화학 결합 [12화학I01-02]

이온 결합(Ionic Bond)

금속 원소 + 비금속 원소의 결합. 전자의 이동으로 형성됨.
  • 금속 원소: 전자를 잃어 양이온이 됨 (예: Na → Na⁺)
  • 비금속 원소: 전자를 얻어 음이온이 됨 (예: Cl + e⁻ → Cl⁻)
  • 양이온과 음이온 사이의 정전기적 인력으로 결합
  • 예: NaCl(소금), MgO, CaCl₂, KBr

공유 결합(Covalent Bond)

비금속 원소 + 비금속 원소의 결합. 전자의 공유로 형성됨.
  • 두 원자가 전자쌍을 서로 공유하여 결합
  • 단일 결합(전자쌍 1쌍), 이중 결합(2쌍), 삼중 결합(3쌍)
  • 예: H₂O, CO₂, NH₃, CH₄, O₂(이중결합), N₂(삼중결합)

5. 화학 반응식 [12화학I02-01]

화학 반응을 원소 기호와 화학식을 이용하여 나타낸 식입니다.

질량 보존 법칙: 화학 반응에서 반응 전후의 원자 종류와 수는 변하지 않음 (질량 보존) → 반응식의 양쪽에서 각 원소의 원자 수가 같아야 함
계수 맞추기 예시

물의 생성 반응: 2H₂ + O₂ → 2H₂O
  (좌: H 4개, O 2개 = 우: H 4개, O 2개 ✓)

메탄의 연소 반응: CH₄ + 2O₂ → CO₂ + 2H₂O
  (좌: C 1, H 4, O 4 = 우: C 1, H 4, O 4 ✓)

철과 산소 반응: 4Fe + 3O₂ → 2Fe₂O₃
  (쇠의 녹스는 반응)

6. 산(Acid)과 염기(Base) [12화학I03-01]

산의 성질

산(Acid): 물에 녹았을 때 수소 이온(H⁺)을 내놓는 물질
  • pH < 7
  • 신맛, 금속과 반응하여 수소 기체 발생
  • 예: 염산(HCl), 황산(H₂SO₄), 질산(HNO₃), 아세트산(CH₃COOH), 탄산(H₂CO₃)
  • 리트머스 종이: 파란색 → 붉은색으로 변화

염기의 성질

염기(Base): 물에 녹았을 때 수산화 이온(OH⁻)을 내놓는 물질
  • pH > 7
  • 쓴맛, 미끄러운 느낌, 단백질 분해
  • 예: 수산화나트륨(NaOH), 수산화칼륨(KOH), 암모니아(NH₃), 수산화칼슘(Ca(OH)₂)
  • 리트머스 종이: 붉은색 → 파란색으로 변화

pH 척도

0-6 산성 7 중성 8-14 염기성
  • pH = 7: 순수한 물 (25°C에서 중성)
  • pH가 낮을수록 강한 산, pH가 높을수록 강한 염기
  • pH 1씩 낮아질 때마다 H⁺ 농도는 10배 증가

중화 반응

산 + 염기 → 염 + 물 (발열 반응)

HCl + NaOH → NaCl + H₂O
H₂SO₄ + 2NaOH → Na₂SO₄ + 2H₂O
(알짜 이온 반응: H⁺ + OH⁻ → H₂O)

7. 산화·환원(Oxidation-Reduction) [12화학I03-02]

산화(Oxidation)와 환원(Reduction)은 항상 동시에 일어남

구분 산소 관점 전자 관점 산화수
산화 산소를 얻음 전자를 잃음 산화수 증가
환원 산소를 잃음 전자를 얻음 산화수 감소
⚠️ 자주 하는 실수

산화 = 산소와 결합? 환원 = 산소 제거? — 이건 산소 기준의 정의입니다. 더 일반적인 정의는 전자 기준: 산화 = 전자 잃음, 환원 = 전자 얻음. 그리고 "LEO GER" 암기법: Loses Electrons = Oxidation, Gains Electrons = Reduction

❌ 흔한 실수 — 산화제·환원제 혼동
산화제는 스스로 산화되는 물질이다 산화제는 스스로 환원되면서 다른 물질을 산화시키는 물질이다
"산화제"라는 이름 때문에 산화제 자신이 산화된다고 착각합니다. 반대입니다: 산화제 자신 → 환원, 환원제 자신 → 산화. 이름과 역할이 반대되므로 암기 필수.
⚠ 자주 틀리는 포인트
  • 이온 결합과 공유 결합 구분: 금속+비금속=이온결합(NaCl), 비금속+비금속=공유결합(H₂O, CO₂). 금속끼리는 금속결합
  • 산화수 계산 오류: H는 +1, O는 -2가 기본. 화합물에서 산화수 합=0. H₂O에서 H(+1)×2+O(-2)=0 확인
  • 몰 계산에서 아보가드로수 적용 오류: 1몰=6.02×10²³개. "분자 수"를 구할 때는 몰 수×6.02×10²³, "질량"을 구할 때는 몰 수×분자량
🧠 산화·환원 암기법

OIL RIG: Oxidation Is Loss (of electrons)   Reduction Is Gain (of electrons)

산화제 vs 환원제: 자신이 환원되면 = 산화제, 자신이 산화되면 = 환원제 (반대!

예제: 마그네슘의 연소 반응 분석
2Mg + O₂ → 2MgO

Mg: 0 → +2 (산화수 증가) → 산화됨
O: 0 → −2 (산화수 감소) → 환원됨

Mg = 환원제 (자신은 산화되면서 O₂를 환원시킴)
O₂ = 산화제 (자신은 환원되면서 Mg를 산화시킴)
예제: 구리의 산화·환원
CuO + H₂ → Cu + H₂O

Cu: +2 → 0 (환원됨) — 산화구리(CuO)가 환원되어 구리(Cu) 생성
H: 0 → +1 (산화됨) — 수소(H₂)가 산화되어 물(H₂O) 형성

    순물질: 금(Au)(원소), 이산화탄소(CO₂)(화합물)
    공기, 소금물, 흙탕물은 혼합물

  2. 원자번호 11번인 나트륨(Na)의 양성자 수, 전자 수, 중성자 수를 구하시오. (질량수 = 23)
    정답 보기

    양성자 수 = 11, 전자 수 = 11, 중성자 수 = 23 − 11 = 12개

  3. 다음 화학 반응식의 빈칸을 채우시오: ___Al + ___O₂ → ___Al₂O₃
    정답 보기
    4Al + 3O₂ → 2Al₂O₃
    (좌: Al 4, O 6 = 우: Al 4, O 6 ✓)
  4. pH 3인 용액과 pH 11인 용액을 중화시켰더니 pH 7이 되었다. 어떤 이온이 반응했는가?
    정답 보기

    H⁺(수소 이온)과 OH⁻(수산화 이온)이 반응: H⁺ + OH⁻ → H₂O
    pH 3 = 산성(H⁺ 많음), pH 11 = 염기성(OH⁻ 많음)

  5. 2Fe + 3Cl₂ → 2FeCl₃ 반응에서 산화된 것과 환원된 것을 구분하시오.
    정답 보기

    Fe: 0 → +3 (산화수 증가) → 산화됨 (Fe = 환원제)
    Cl: 0 → −1 (산화수 감소) → 환원됨 (Cl₂ = 산화제)

💡 화학 공부 핵심: 화학 반응식을 쓸 때 반드시 원자 수 균형을 확인하세요. 산화·환원 반응은 "산화수의 증가/감소"로 판단하는 습관을 들이세요.
🔗 다음으로 가는 다리

원자와 분자가 결합해 만든 유기물이 어떻게 살아있는 세포가 될까?

세포를 구성하는 단백질·DNA·지질은 모두 탄소 기반 유기화합물이다. 공유결합·이온결합으로 배운 원리가 생명체의 분자 기계를 설명한다.

생물로 이동
🔓 이 개념을 마스터하면 해금됩니다
생물학 — 세포·DNA의 화학적 구조 원자와 분자 심화 — 분자 구조·결합 에너지

화학은 생명과학·의약·재료공학의 공통 언어입니다. 주기율표와 반응식을 이해한 지금, 세포 단위의 화학 반응인 생화학까지 연결할 수 있습니다.

시험 직전 5분 체크리스트
  • 화학 반응식: 반응물 = 생성물 (원자 수 보존)
  • 몰(mol): 1mol = 6.02×10²³ 개 (아보가드로 수)
  • 산과 염기: pH 7 중성, 7 미만 산성, 7 초과 염기성
  • 산화·환원: 산화 = 전자 잃음 / 환원 = 전자 얻음 (항상 동시)
