1. 삼각함수의 정의 (직각삼각형) [12수I02-01]

🧠 기호 없이 직관으로

삼각함수 = 원 위를 걸어다닐 때의 좌표. 반지름 1인 원에서 각도 θ만큼 회전한 점의 x좌표가 cosθ, y좌표가 sinθ입니다. 직각삼각형에서 외우는 것보다 "단위원 위의 점"으로 이해하면 모든 삼각함수 공식이 자연스럽게 연결됩니다.

직각삼각형에서 각 \(\theta\)에 대해 삼각비를 정의합니다.

삼각비 정의 \[ \sin\theta = \frac{\text{대변(맞은편 변)}}{\text{빗변}} \] \[ \cos\theta = \frac{\text{밑변(인접한 변)}}{\text{빗변}} \] \[ \tan\theta = \frac{\text{대변}}{\text{밑변}} = \frac{\sin\theta}{\cos\theta} \]

주요 각도의 삼각함수 값

각도 \(\theta\) 라디안 \(\sin\theta\) \(\cos\theta\) \(\tan\theta\)
0° 0 0 1 0
30° \(\dfrac{\pi}{6}\) \(\dfrac{1}{2}\) \(\dfrac{\sqrt{3}}{2}\) \(\dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}\)
45° \(\dfrac{\pi}{4}\) \(\dfrac{\sqrt{2}}{2}\) \(\dfrac{\sqrt{2}}{2}\) 1
60° \(\dfrac{\pi}{3}\) \(\dfrac{\sqrt{3}}{2}\) \(\dfrac{1}{2}\) \(\sqrt{3}\)
90° \(\dfrac{\pi}{2}\) 1 0 정의 안됨
180° \(\pi\) 0 −1 0
sin·cos·tan 암기팁"sin = 높이/빗변, cos = 밑변/빗변, tan = 높이/밑변" — 분자는 sin이 대변, cos이 인접변. tan = sin÷cos
특수각 값 암기팁sin 0°→90°: 0, ½, √2/2, √3/2, 1 — 분자가 √0, √1, √2, √3, √4를 2로 나눈 것. cos는 거꾸로!
✏️ 바로 확인 — 특수각

문제: sin 60° × cos 30°의 값은?

정답 확인하기
\(\sin 60° = \dfrac{\sqrt{3}}{2}\), \(\cos 30° = \dfrac{\sqrt{3}}{2}\) → \(\dfrac{\sqrt{3}}{2} \times \dfrac{\sqrt{3}}{2} = \dfrac{3}{4}\)

2. 단위원과 삼각함수

반지름이 1인 원(단위원)을 이용해 삼각함수를 모든 각도로 확장할 수 있습니다.

단위원에서의 정의

단위원 위의 점 \(P(x, y)\)와 x축 사이의 각이 \(\theta\)일 때:

\[ \cos\theta = x, \quad \sin\theta = y, \quad \tan\theta = \frac{y}{x} \]

피타고라스 항등식

\[ \sin^2\theta + \cos^2\theta = 1 \]

이로부터 다음을 유도할 수 있습니다:

\[ \tan^2\theta + 1 = \sec^2\theta \qquad (1 + \cot^2\theta = \csc^2\theta) \]

사분면에서의 부호

사분면sincostan
제1사분면 (0° ~ 90°)+++
제2사분면 (90° ~ 180°)+−−
제3사분면 (180° ~ 270°)−−+
제4사분면 (270° ~ 360°)−+−

3. 삼각함수의 그래프

\(y = \sin x\)

  • 주기: \(2\pi\) (360°)
  • 최댓값: 1, 최솟값: −1 (진폭: 1)
  • 원점 대칭 (기함수): \(\sin(-x) = -\sin x\)
  • 영점(0이 되는 x값): \(x = n\pi\) (\(n\)은 정수)

\(y = \cos x\)

  • 주기: \(2\pi\)
  • 최댓값: 1, 최솟값: −1
  • y축 대칭 (우함수): \(\cos(-x) = \cos x\)
  • \(y = \sin x\)를 \(\dfrac{\pi}{2}\)만큼 왼쪽으로 이동한 것

\(y = \tan x\)

  • 주기: \(\pi\) (180°)
  • 점근선: \(x = \dfrac{\pi}{2} + n\pi\) (\(n\)은 정수)
  • 범위: 실수 전체 (값의 범위에 제한 없음)

일반적인 삼각함수 \(y = A\sin(Bx + C) + D\)

  • 진폭(Amplitude) = \(|A|\)
  • 주기(Period) = \(\dfrac{2\pi}{|B|}\)
  • 위상(Phase shift) = \(-\dfrac{C}{B}\)
  • 수직 이동 = \(D\)

4. 삼각함수의 주요 공식 [12수I02-03]

피타고라스 항등식 (재확인)

\[ \sin^2\theta + \cos^2\theta = 1 \]

덧셈정리

\begin{align} \sin(\alpha \pm \beta) &= \sin\alpha\cos\beta \pm \cos\alpha\sin\beta \\[0.5em] \cos(\alpha \pm \beta) &= \cos\alpha\cos\beta \mp \sin\alpha\sin\beta \\[0.5em] \tan(\alpha + \beta) &= \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta} \end{align}
예제: \(\sin 75°\) 계산하기
\(\sin 75° = \sin(45° + 30°)\)
\(= \sin 45°\cos 30° + \cos 45°\sin 30°\)
\(= \dfrac{\sqrt{2}}{2} \cdot \dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2} \cdot \dfrac{1}{2}\)
\(= \dfrac{\sqrt{6}}{4} + \dfrac{\sqrt{2}}{4} = \dfrac{\sqrt{6}+\sqrt{2}}{4}\)

배각 공식

\begin{align} \sin 2\theta &= 2\sin\theta\cos\theta \\[0.5em] \cos 2\theta &= \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta \\[0.5em] \tan 2\theta &= \frac{2\tan\theta}{1 - \tan^2\theta} \end{align}

반각 공식

\begin{align} \sin^2\frac{\theta}{2} &= \frac{1 - \cos\theta}{2} \\[0.5em] \cos^2\frac{\theta}{2} &= \frac{1 + \cos\theta}{2} \end{align}
⚠️ 수능·내신 자주 틀리는 포인트
  • cos0° = 1, cos90° = 0 혼동 — "cos는 0에서 1, sin은 0에서 시작해 90°에서 1" — 헷갈리면 단위원 그리기
  • 사인법칙 vs 코사인법칙 선택 실수 — 두 각 + 한 변: 사인법칙 / 두 변 + 끼인각, 세 변: 코사인법칙
  • 덧셈정리 부호 실수 — sin(A−B) = sinAcosB − cosAsinB (빼기 문제에서 더하기로 실수)

5. 사인 법칙과 코사인 법칙

삼각형 ABC에서 변 a, b, c는 각 A, B, C의 대변(맞은편 변)이고, R은 외접원의 반지름입니다.

사인 법칙

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \]

두 각과 한 변을 알 때, 또는 외접원의 반지름을 구할 때 사용합니다.

예제: A = 30°, B = 45°, c = 6일 때 a 구하기
C = 180° − 30° − 45° = 105°
사인법칙: \(\dfrac{a}{\sin 30°} = \dfrac{6}{\sin 105°}\)
\(a = \dfrac{6\sin 30°}{\sin 105°} = \dfrac{6 \times \frac{1}{2}}{\frac{\sqrt{6}+\sqrt{2}}{4}} = \dfrac{3 \times 4}{\sqrt{6}+\sqrt{2}} = \dfrac{12}{\sqrt{6}+\sqrt{2}}\)

코사인 법칙

\begin{align} a^2 &= b^2 + c^2 - 2bc\cos A \\[0.5em] b^2 &= a^2 + c^2 - 2ac\cos B \\[0.5em] c^2 &= a^2 + b^2 - 2ab\cos C \end{align}

두 변과 끼인각, 또는 세 변을 알 때 사용합니다.

예제: a = 5, b = 7, C = 60°일 때 c 구하기
\(c^2 = 5^2 + 7^2 - 2 \cdot 5 \cdot 7 \cdot \cos 60°\)
\(= 25 + 49 - 70 \times \dfrac{1}{2}\)
\(= 74 - 35 = 39\)
\(c = \sqrt{39}\)

6. 연습 문제

  1. ⭐ 기초 \(\sin 30° \times \cos 60° + \cos 30° \times \sin 60°\)를 계산하시오.
    정답 보기
    \(\sin(30°+60°) = \sin 90° = \mathbf{1}\) (덧셈정리 역이용)
  2. ⭐⭐ 표준 \(\sin\theta = \dfrac{3}{5}\) (단, \(0° < \theta < 90°\))일 때, \(\cos\theta\)와 \(\tan\theta\)를 구하시오.
    정답 보기
    \(\cos^2\theta = 1 - \dfrac{9}{25} = \dfrac{16}{25}\) → \(\cos\theta = \dfrac{4}{5}\)  ·  \(\tan\theta = \dfrac{3/5}{4/5} = \mathbf{\dfrac{3}{4}}\)
  3. ⭐⭐ 표준 배각 공식을 이용하여 \(\cos 2\theta\)를 \(\sin\theta\)만으로 나타내시오.
    정답 보기
    \(\cos 2\theta = 1 - 2\sin^2\theta\)
  4. ⭐⭐ 표준 삼각형 ABC에서 a = 8, b = 6, C = 90°일 때, c와 \(\cos A\)를 구하시오.
    정답 보기
    \(c = \sqrt{8^2+6^2} = 10\)  ·  \(\cos A = \dfrac{6^2+10^2-8^2}{2\cdot6\cdot10} = \dfrac{72}{120} = \mathbf{\dfrac{3}{5}}\)
  5. ⭐⭐⭐ 심화 \(\cos 120°\)의 값을 덧셈정리를 이용하여 구하시오.
    정답 보기
    \(\cos(90°+30°) = -\sin 30° = \mathbf{-\dfrac{1}{2}}\)
