🌱 왜 방정식을 배우는가?

방정식은 "모르는 수를 찾는" 수학의 핵심 도구입니다. 물건 가격, 두 기차의 만남, 최적 생산량 — 현실의 수많은 문제는 미지수가 있는 조건으로 표현됩니다. 방정식은 그 조건을 수식으로 쓰고 미지수를 구하는 언어입니다. 이후 배울 함수·삼각함수·미적분은 모두 방정식 풀이 위에서 성립하며, 이차방정식의 근의 공식은 대학 수학·공학·경제학 전반에서 반복적으로 등장합니다.

💡 이걸로 이해하면 됩니다
방정식 = 양팔 저울

양팔 저울에서 왼쪽과 오른쪽이 균형을 이룬다. 양쪽에 같은 것을 더하거나 빼거나 곱해도 균형은 유지된다. 이항은 한쪽 물건을 반대편 저울로 옮기되, 부호(+/-)를 뒤집는 것이다 — 저울을 기울이지 않으려면 반대편에 넣을 때 "마이너스"로 변환해야 한다.

⚡ 핵심 30초 요약
  • 방정식: 미지수가 있는 등식 — 참이 되게 하는 값이 해(근)
  • 일차방정식: \(ax + b = 0\) → 이항하여 \(x = -b/a\)
  • 연립방정식: 미지수 2개 · 방정식 2개 → 가감법 또는 대입법
  • 이차방정식: \(ax^2 + bx + c = 0\) → 인수분해 또는 근의 공식
  • 판별식 \(D = b^2 - 4ac\): \(D>0\) 실근 2개, \(D=0\) 중근, \(D<0\) 허근
학습자 질문

방정식과 항등식은 어떻게 다른가요?

방정식은 특정 x값에서만 성립하는 조건부 등식입니다. x + 1 = 5는 x = 4일 때만 참. 항등식은 어떤 x를 넣어도 성립합니다 — 2(x+1) = 2x+2는 항상 참. 참이 되는 x의 범위가 핵심 차이입니다.

1. 방정식이란?

방정식(方程式)은 미지수(변수)를 포함한 등식으로, 특정 값에서만 참이 됩니다. 방정식을 만족시키는 값을 해(解) 또는 근(根)이라 하고, 해를 구하는 과정을 방정식을 푼다고 합니다.

방정식과 항등식의 비교
  • 방정식: 특정 값에서만 성립 → 예: \(x + 3 = 7\) (해: \(x = 4\)일 때만 성립)
  • 항등식: 모든 값에서 성립 → 예: \(2(x+1) = 2x + 2\) (x에 어떤 수를 대입해도 성립)

2. 일차방정식 [09수02-04]

미지수의 최고 차수가 1인 방정식입니다.

\[ ax + b = 0 \quad (a \neq 0) \]

이항(移項)

등호의 한쪽에 있는 항을 부호를 바꾸어 반대쪽으로 옮기는 것을 이항이라 합니다.
원리: 등호 양변에 같은 수를 더하거나 빼어도 등식이 성립합니다.

🔎 이 공식은 어디서 왔나?
근의 공식 유도 (완전제곱식 변환)
ax²+bx+c=0을 x에 대해 풀어보면:
① 양변을 a로 나눔: x²+(b/a)x+c/a=0
② 완전제곱식으로 변환: (x+b/2a)²= b²/4a²−c/a
③ 오른쪽 통분: (b²−4ac)/4a²
④ 제곱근 취함: x = (−b ± √(b²−4ac)) / 2a — 이것이 근의 공식
이항 암기팁"부호를 바꿔서 반대편으로" — 더하기는 건너가면 빼기, 곱하기는 건너가면 나누기. 검산은 필수: 구한 x를 원래 식에 대입해 확인
🔢 단계별 풀이 — 3x + 5 = 14
  1. 상수 이항우변에 있는 상수 5를 좌변으로: 3x = 14 − 5 = 9
  2. 계수로 나누기양변을 계수 3으로 나눔: x = 9 ÷ 3
  3. 해 계산x = 3
  4. 검산3(3) + 5 = 9 + 5 = 14 ✓
✏️ 바로 확인 — 일차방정식

문제: 2x − 3 = 9를 풀어 x를 구하시오.

정답 확인하기
2x = 9 + 3 = 12 → x = 6   검산: 2(6)−3 = 9 ✓
예제: 2x − 7 = 3x + 1 풀기
2x − 7 = 3x + 1
2x − 3x = 1 + 7   (미지수 항은 좌변, 상수 항은 우변으로 이항)
−x = 8
x = −8
예제: 괄호가 있는 방정식 5(x − 2) = 3x + 4 풀기
5x − 10 = 3x + 4   (분배법칙으로 괄호 전개)
5x − 3x = 4 + 10
2x = 14
x = 7

3. 연립일차방정식

두 개의 미지수(x, y)를 포함한 두 일차방정식의 쌍입니다. 두 방정식을 동시에 만족하는 \((x, y)\)를 구합니다.

가감법 (덧셈/뺄셈법)

두 방정식을 더하거나 빼서 미지수를 하나 소거합니다.

예제: 가감법으로 풀기 \[ \begin{cases} 2x + y = 5 \\ x - y = 1 \end{cases} \]
두 식을 더함: (2x + y) + (x − y) = 5 + 1
3x = 6 → x = 2
x = 2를 두 번째 식에 대입: 2 − y = 1 → y = 1
해: \((x, y) = (2, 1)\)
검산: 2(2)+1=5 ✓, 2−1=1 ✓

대입법

한 방정식에서 한 미지수를 다른 미지수로 표현하여 다른 방정식에 대입합니다.

예제: 대입법으로 풀기 \[ \begin{cases} y = 2x - 1 \\ 3x + 2y = 13 \end{cases} \]
첫 번째 식을 두 번째 식에 대입:
3x + 2(2x − 1) = 13
3x + 4x − 2 = 13
7x = 15 → x = 15/7... (정수해 없음)

다시 정수해 예제: \(\begin{cases} y = x + 3 \\ 2x + y = 9 \end{cases}\)
대입: 2x + (x + 3) = 9 → 3x = 6 → x = 2, y = 5

4. 이차방정식 [09수02-06]

미지수의 최고 차수가 2인 방정식입니다.

\[ ax^2 + bx + c = 0 \quad (a \neq 0) \]

인수분해를 이용한 풀이

\(AB = 0\)이면 \(A = 0\) 또는 \(B = 0\)임을 이용합니다.

예제: \(x^2 - x - 6 = 0\) 풀기
인수분해: \((x + 2)(x - 3) = 0\)
\(x + 2 = 0\) 또는 \(x - 3 = 0\)
\(x = -2\) 또는 \(x = 3\)

근의 공식

인수분해가 되지 않는 경우 근의 공식을 사용합니다.

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]
근의 공식 암기팁"마이너스 b에 플마 루트 b제곱 마이너스 4ac, 2a로 나눈다" — a, b, c는 ax²+bx+c=0에서 각 항의 계수
예제: \(2x^2 - 5x + 2 = 0\) 풀기 (a=2, b=−5, c=2)
\(x = \dfrac{-(-5) \pm \sqrt{(-5)^2 - 4 \cdot 2 \cdot 2}}{2 \cdot 2}\)

\(x = \dfrac{5 \pm \sqrt{25 - 16}}{4} = \dfrac{5 \pm \sqrt{9}}{4} = \dfrac{5 \pm 3}{4}\)

\(x = \dfrac{5+3}{4} = 2\)   또는   \(x = \dfrac{5-3}{4} = \dfrac{1}{2}\)
∴ \(x = 2\) 또는 \(x = \dfrac{1}{2}\)
⚖️ 풀이법 비교 — 인수분해 vs 근의 공식
🔷 인수분해
  • 두 수의 합 = b, 곱 = c인 수를 찾는다
  • 계산이 빠르고 간단
  • 정수 범위에서 인수가 있을 때 사용
🔶 근의 공식
  • a, b, c를 대입해 계산
  • 항상 사용 가능하지만 계산량 많음
  • 인수분해가 안 될 때 필수
선택 기준: 먼저 인수분해 시도 → 30초 내에 안 되면 근의 공식. 판별식 D<0이면 실근 없음.

판별식 D

\[ D = b^2 - 4ac \]
  • \(D > 0\): 서로 다른 두 실근
  • \(D = 0\): 중근 (서로 같은 두 실근)
  • \(D < 0\): 실근 없음 (허근)
⚠️ 수능·내신 자주 틀리는 포인트
  • 이항 시 부호 바꾸는 것 잊기 — \(3x + 5 = 14\)에서 5를 이항하면 반드시 \(-5\). "이항 = 부호 반전"
  • 양변에 음수 곱할 때 부등호 방향 — 방정식은 괜찮지만 부등식은 음수 곱하면 부등호 방향 바뀜
  • 0으로 나누는 실수 — 이차방정식 \(ax^2 + bx = 0\) 풀 때 양변을 x로 나누면 x=0 해를 잃음

5. 방정식의 활용

문장제 문제를 방정식으로 풀 때의 순서입니다.

문장제 풀이 순서
  1. 구하는 것을 미지수 \(x\)로 놓습니다.
  2. 조건을 방정식으로 세웁니다.
  3. 방정식을 풉니다.
  4. 답이 문제의 조건(자연수, 양수 등)에 맞는지 확인합니다.
예제: 어떤 수의 3배에서 7을 빼면 20이 됩니다. 어떤 수를 구하시오.
어떤 수를 \(x\)라 하면:
3x − 7 = 20
3x = 27
x = 9

6. 연습 문제

  1. ⭐ 기초 \(4x - 9 = 11\)을 풀어라.
    정답 보기
    \(4x = 20\) → \(x = 5\)
  2. ⭐⭐ 표준 \(\begin{cases} x + 2y = 8 \\ 2x - y = 6 \end{cases}\)을 풀어라.
    정답 보기
    ② ×2 → \(4x-2y=12\). ①+: \(5x=20\) → \(x=4\), \(y=2\)   ∴ (4, 2)
  3. ⭐⭐ 표준 \(x^2 + x - 12 = 0\)을 인수분해로 풀어라.
    정답 보기
    \((x+4)(x-3)=0\) → \(x=-4\) 또는 \(x=3\)
  4. ⭐⭐ 표준 근의 공식을 이용하여 \(x^2 - 4x + 1 = 0\)을 풀어라.
    정답 보기
    \(x = \dfrac{4 \pm \sqrt{16-4}}{2} = \dfrac{4 \pm \sqrt{12}}{2} = \mathbf{2 \pm \sqrt{3}}\)
  5. ⭐⭐⭐ 심화 연속하는 두 자연수의 합이 31입니다. 두 수를 구하시오.
    정답 보기
    두 수를 \(n, n+1\)로 놓으면 \(2n+1=31\) → \(n=15\)   ∴ 15와 16
