🌿 生態系完全まとめ

📚 中学・高校 生物 🎯 重要概念：食物連鎖・エネルギー流・物質循環・生物多様性

1. 生態系の構成

生態系(Ecosystem)：生物的要素（生産者・消費者・分解者）＋非生物的要素（光・温度・水・土壌）
  • 生産者(Producer)：光合成で有機物を合成。植物・藻類
  • 消費者(Consumer)：一次消費者（草食動物）→ 二次消費者（肉食動物）→ 三次消費者
  • 分解者(Decomposer)：死んだ有機物を分解。細菌・菌類

2. 食物連鎖と食物網

食物連鎖(Food Chain)：草 → バッタ → カエル → ヘビ → タカ

食物網(Food Web)：複数の食物連鎖が複雑に絡み合った構造 → 生態系の安定性を高める

生態ピラミッド：上位の栄養段階に行くほど個体数・エネルギー・生体量が減少

3. エネルギー流

エネルギーは太陽 → 生産者 → 消費者 → 分解者の順に流れ、各段階で約10%だけが次の段階へ伝達（残り90%は熱として散逸）。

エネルギーの流れは一方向：物質循環と異なり、循環しない。

例題：生産者が1000 kcalのエネルギーを持つとき、二次消費者（肉食動物）が得られる最大エネルギーはいくらか？
解答：生産者(1000) → 一次消費者(10%) = 100 kcal → 二次消費者(10%) = 10 kcal

4. 物質循環

  • 炭素循環：光合成（CO₂吸収）↔ 呼吸・燃焼（CO₂放出）。生物圏↔大気圏の循環
  • 窒素循環：窒素固定（N₂→アンモニア）→ 硝化 → 植物吸収 → 分解・脱窒
  • 水の循環：蒸発 → 降水 → 地表水・地下水 → 再蒸発

5. 生物多様性

  • 遺伝的多様性：同じ種内の遺伝子の違い → 環境変化への適応力
  • 種の多様性：ある地域の種数とその分布
  • 生態系の多様性：森林・湿地・サンゴ礁など多様な生息環境
  • 脅威の要因：生息地の破壊、気候変動、外来種の侵入、乱獲

練習問題

問題：生産者が100kcalのエネルギーを持つとき、二次消費者（肉食動物）が得られるエネルギーは最大いくらか？
解答：生産者(100) → 一次消費者(10%) = 10kcal → 二次消費者(10%) = 1kcal
💡 学習のヒント：エネルギーの流れは一方向、物質循環は循環するという違いを必ず区別しましょう。生態ピラミッドの計算は常に「10%法則」を適用。生物多様性の減少原因（生息地破壊・気候変動・外来種・乱獲）も試験頻出です。
