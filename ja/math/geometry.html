📐 図形の完全解説

📚 中学数学 🎯 重要概念：三角形、四角形、円、三平方の定理

1. 三角形

三角形の内角の和は必ず180°です。

三角形の面積と周 \[ S = \frac{1}{2} \times \text{底辺} \times \text{高さ} \] \[ \text{周} = a + b + c \]
三角形の種類（辺）
  • 正三角形：3辺すべて等しい。3つの角すべて60°
  • 二等辺三角形：2辺が等しい。底角が等しい
  • 不等辺三角形：3辺すべて異なる
三角形の種類（角）
  • 鋭角三角形：3つの内角すべて90°未満
  • 直角三角形：1つの内角が90°
  • 鈍角三角形：1つの内角が90°超

2. 三平方の定理（ピタゴラスの定理）

直角三角形において、直角を挟む2辺の長さの2乗の和は斜辺の長さの2乗に等しい。

\[ a^2 + b^2 = c^2 \]

a・b：直角を挟む2辺（脚）、c：斜辺（最も長い辺）

例1：斜辺を求める（a=3, b=4）
\(c^2 = 3^2 + 4^2 = 9 + 16 = 25 \Rightarrow c = 5\)
例2：脚を求める（c=13, a=5）
\(b^2 = 13^2 - 5^2 = 169 - 25 = 144 \Rightarrow b = 12\)
よく使うピタゴラス数： (3,4,5)、(5,12,13)、(8,15,17)、(6,8,10)など。これらの整数倍も使えます。

3. 四角形の性質と面積

面積の公式
  • 長方形：\(S = l \times w\)（l：縦、w：横）
  • 正方形：\(S = a^2\)（a：1辺）
  • 平行四辺形：\(S = \text{底辺} \times \text{高さ}\)
  • 台形：\(S = \dfrac{1}{2}(\text{上底} + \text{下底}) \times \text{高さ}\)
  • ひし形：\(S = \dfrac{d_1 \times d_2}{2}\)（d：対角線）

4. 円

  • 半径（r）：中心から円周までの距離
  • 直径（d）：\(d = 2r\)
  • 円周（C）：円の周の長さ
  • 円周率（π）≈ 3.14159
\[ C = 2\pi r = \pi d, \qquad S = \pi r^2 \]
例：半径7cmの円
\(C = 2\pi \times 7 = 14\pi \approx 44.0\) cm
\(S = \pi \times 49 = 49\pi \approx 153.9\) cm²

5. 練習問題

  1. 底辺8cm、高さ5cmの三角形の面積を求めなさい。
  2. 直角三角形の2辺が6cmと8cmのとき、斜辺は何cmですか？
  3. 半径5cmの円の面積と円周を求めなさい（π=3.14）。
  4. 上底4cm、下底10cm、高さ6cmの台形の面積を求めなさい。
答え
  1. \(S = \dfrac{1}{2} \times 8 \times 5 = 20\) cm²
  2. \(c = \sqrt{36+64} = \sqrt{100} = 10\) cm
  3. \(S = 3.14 \times 25 = 78.5\) cm²、\(C = 2 \times 3.14 \times 5 = 31.4\) cm
  4. \(S = \dfrac{1}{2}(4+10) \times 6 = 42\) cm²
