📐 図形の完全解説

📚 中学数学 🎯 重要概念：三角形、四角形、円、三平方の定理

1. 三角形

三角形の内角の和は必ず180°です。

三角形の面積と周 \[ S = \frac{1}{2} \times \text{底辺} \times \text{高さ} \] \[ \text{周} = a + b + c \]
三角形の種類（辺）
  • 正三角形：3辺すべて等しい。3つの角すべて60°
  • 二等辺三角形：2辺が等しい。底角が等しい
  • 不等辺三角形：3辺すべて異なる
三角形の種類（角）
  • 鋭角三角形：3つの内角すべて90°未満
  • 直角三角形：1つの内角が90°
  • 鈍角三角形：1つの内角が90°超

2. 三平方の定理（ピタゴラスの定理）

🏛 概念の誕生

この定理はピタゴラス（紀元前570年頃）の名で知られていますが、バビロニアの粘土板（紀元前1800年）にも記録されており、中国の周髀算経（紀元前1000年頃）にも登場します。ピタゴラスが初めて体系的な証明を与えたとされています。この定理は数学史上600種類以上の異なる証明が存在し、アメリカ第20代大統領ガーフィールドも独自の証明を発表しました。

直角三角形において、直角を挟む2辺の長さの2乗の和は斜辺の長さの2乗に等しい。

\[ a^2 + b^2 = c^2 \]

a・b：直角を挟む2辺（脚）、c：斜辺（最も長い辺）

例1：斜辺を求める（a=3, b=4）
\(c^2 = 3^2 + 4^2 = 9 + 16 = 25 \Rightarrow c = 5\)
例2：脚を求める（c=13, a=5）
\(b^2 = 13^2 - 5^2 = 169 - 25 = 144 \Rightarrow b = 12\)
よく使うピタゴラス数： (3,4,5)、(5,12,13)、(8,15,17)、(6,8,10)など。これらの整数倍も使えます。
⚠️ よくある間違い

三平方の定理はすべての三角形に使える？ — いいえ。\(a^2 + b^2 = c^2\) は直角三角形専用です。直角（90°）があるかどうか必ず確認してから使いましょう。

✏️ 確認クイズ

直角三角形の2辺が6cmと8cmのとき、斜辺を求めなさい。

▶ 答えを見る
\(c^2 = 6^2 + 8^2 = 36 + 64 = 100\) → \(c = \mathbf{10\text{ cm}}\)  （3-4-5の2倍組合せ）

3. 四角形の性質と面積

面積の公式
  • 長方形：\(S = l \times w\)（l：縦、w：横）
  • 正方形：\(S = a^2\)（a：1辺）
  • 平行四辺形：\(S = \text{底辺} \times \text{高さ}\)
  • 台形：\(S = \dfrac{1}{2}(\text{上底} + \text{下底}) \times \text{高さ}\)
  • ひし形：\(S = \dfrac{d_1 \times d_2}{2}\)（d：対角線）

4. 円

  • 半径（r）：中心から円周までの距離
  • 直径（d）：\(d = 2r\)
  • 円周（C）：円の周の長さ
  • 円周率（π）≈ 3.14159
\[ C = 2\pi r = \pi d, \qquad S = \pi r^2 \]
例：半径7cmの円
\(C = 2\pi \times 7 = 14\pi \approx 44.0\) cm
\(S = \pi \times 49 = 49\pi \approx 153.9\) cm²
❌ よくある間違い — 面積と円周の混同
半径rの円の面積 = 2πr（円周の公式を代入してしまう） 面積 = πr²（単位：cm²）、円周 = 2πr（単位：cm）
面積はr²（2次元）、円周はr（1次元）。単位で判断できます：cm²なら面積、cmなら円周。
🧠 暗記ポイント

円の公式2つ: 面積はr²（πr²）、周はr¹（2πr）— 指数が次元を表す！

円柱 vs 円錐: 同じ底面・同じ高さなら 円錐の体積 = 円柱の 1/3

頻出ピタゴラス数: 3-4-5、5-12-13 は試験に必ず出る — 丸暗記！

5. 練習問題

★☆☆ 基本

Q1. 底辺8cm、高さ5cmの三角形の面積を求めなさい。

▶ 答えを見る
\(S = \dfrac{1}{2} \times 8 \times 5 = \mathbf{20\text{ cm}^2}\)
★★☆ 中級

Q2. 半径5cmの円の面積と円周を求めなさい（π=3.14）。

▶ 答えを見る
面積: \(3.14 \times 5^2 = \mathbf{78.5\text{ cm}^2}\)   円周: \(2 \times 3.14 \times 5 = \mathbf{31.4\text{ cm}}\)
★★☆ 中級

Q3. 上底4cm、下底10cm、高さ6cmの台形の面積を求めなさい。

▶ 答えを見る
\(S = \dfrac{1}{2}(4+10) \times 6 = \mathbf{42\text{ cm}^2}\)
