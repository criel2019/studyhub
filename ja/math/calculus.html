∫ 微積分の完全解説

📚 高校数学Ⅱ・Ⅲ 🎯 重要概念：極限、微分、積分

1. 極限

関数 \(f(x)\) においてxをaに近づけたときのfの値が近づく値を極限値といいます。

\[ \lim_{x \to a} f(x) = L \]
例：\(\displaystyle\lim_{x \to 2} (3x + 1)\)
直接代入できる場合：\(3 \times 2 + 1 = 7\)

例（不定形）：\(\displaystyle\lim_{x \to 2} \dfrac{x^2 - 4}{x - 2}\)
\(\dfrac{(x+2)(x-2)}{x-2} = x+2 \to 4\)（x=2を代入）

2. 微分

微分は関数の変化の割合（瞬間の速さ）を表します。幾何学的には接線の傾きです。

\[ f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h} \]

微分の公式

  • 定数： \((c)' = 0\)
  • べき乗則： \((x^n)' = nx^{n-1}\)
  • 和・差： \((f \pm g)' = f' \pm g'\)
  • 積の微分： \((fg)' = f'g + fg'\)
  • 商の微分： \(\left(\dfrac{f}{g}\right)' = \dfrac{f'g - fg'}{g^2}\)
  • 合成関数の微分（連鎖律）： \(\{f(g(x))\}' = f'(g(x)) \cdot g'(x)\)
例：\(f(x) = 3x^4 - 2x^2 + 5x - 1\) の微分
\(f'(x) = 12x^3 - 4x + 5\)
例（合成関数）：\(y = (2x+1)^5\) の微分
\(y' = 5(2x+1)^4 \cdot 2 = 10(2x+1)^4\)

3. 積分

積分は微分の逆演算です。不定積分は \(F'(x) = f(x)\) を満たす \(F(x)+C\)（Cは積分定数）です。

積分の基本公式

  • \(\displaystyle\int x^n dx = \dfrac{x^{n+1}}{n+1} + C \quad (n \neq -1)\)
  • \(\displaystyle\int e^x dx = e^x + C\)
  • \(\displaystyle\int \cos x\, dx = \sin x + C\)
  • \(\displaystyle\int \sin x\, dx = -\cos x + C\)

定積分

\[ \int_a^b f(x)\, dx = [F(x)]_a^b = F(b) - F(a) \]

定積分はグラフとx軸の間の面積を表します。

例：\(\displaystyle\int_1^3 (2x + 1)\, dx\)
\(F(x) = x^2 + x\)
\([x^2 + x]_1^3 = (9+3) - (1+1) = 12 - 2 = 10\)

4. 練習問題

  1. \(f(x) = 5x^3 - 4x^2 + 3x\) を微分しなさい。
  2. \(g(x) = (3x-1)^4\) を微分しなさい。
  3. \(\displaystyle\int (4x^3 - 6x + 2)\, dx\) を求めなさい。
  4. \(\displaystyle\int_0^2 (3x^2 - 2x)\, dx\) を求めなさい。
答え
  1. \(f'(x) = 15x^2 - 8x + 3\)
  2. \(g'(x) = 4(3x-1)^3 \times 3 = 12(3x-1)^3\)
  3. \(x^4 - 3x^2 + 2x + C\)
  4. \([x^3 - x^2]_0^2 = (8-4) - 0 = 4\)
