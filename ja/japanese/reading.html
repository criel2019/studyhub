📖 読解完全まとめ

📚 中学・高校 国語 🎯 重要戦略：主題把握・構造分析・推論・批判的読解

1. 読解とは

読解(読解)は単に文字を読むことではなく、文章の意味を把握し批判的に思考するプロセスです。
  • 事実的読解：文章に明示されている情報を正確に把握
  • 推論的読解：文章に隠された意味や意図を推論
  • 批判的読解：文章の妥当性・信頼性・論理性を評価
  • 鑑賞的読解：文学作品の情感・美的価値を鑑賞

2. 主題・要旨の把握

  • 主題：文章全体が伝えたい核心メッセージ（一文で要約）
  • 中心文：主題を直接表す文。通常は段落の最初か最後に位置
  • キーワード：文章中に繰り返されるキーワードが主題と連結

戦略：各段落の中心内容を一行で整理 → 全体の流れを把握 → 主題を導出

3. 文章構造の分析

論説文・説明文の構造
  • 頭括型：主題文 → 根拠・説明（冒頭に要点）
  • 尾括型：根拠・説明 → 主題文（末尾に要点）
  • 双括型：主題文 → 本論 → 主題文の再強調
接続語で構造を把握
  • しかし・だが → 逆接（転換）
  • したがって・そのため → 結論
  • たとえば・つまり → 補足・例示
  • また・さらに → 添加

4. 推論と批判的読解

推論の方法
  • 語彙の意味推論：文脈から知らない語の意味を把握
  • 隠れた意図の推論：筆者の観点・目的を把握
  • 空欄推論：前後の文脈から空欄に適切な内容を推論
批判的読解チェック
  • 根拠は妥当か？出典は信頼できるか？
  • 論理的誤謬（早まった一般化・無関係な権威など）はないか？
  • 別の観点はないか？

練習問題

次の文章を読んで、主題を一文で書きなさい。
「現代人は一日平均11時間以上スクリーンを見つめている。過度なデジタル接触は睡眠の質を低下させ、集中力を妨げる。したがって、一日1時間以上の『デジタルデトックス』の時間を設けることが健全な生活に必要だ。」
解答例：健全な生活のためにデジタルデトックスの時間を設けることが必要だ。
💡 学習のヒント：読解力はさまざまな種類の文章を毎日コツコツ読むことで向上します。新聞の社説・科学雑誌・文学作品などを読み、各段落の核心を一行で整理する練習をしましょう。共通テストの現代文は速度と正確性が命です。
