🌐 英語学習ノート

単語暗記より先に — 文法の原理を知れば、知らない文も解読できる

英語の試験で失点する問題の多くは、語彙不足ではなく文の構造が読めないからです。関係詞・仮定法・分詞構文——この3つの構造を原理から理解すれば、長くて複雑な英文も区切って読めるようになります。
📍 初めての方はこの順で
英文法› 語彙› 読解› 英作文 · 配点の高い順に構成
💡 英語学習のポイント: 文法の理解が読解力と英作文力の向上につながります。語彙を増やしながら様々な英文を繰り返し読むことが最も効果的な学習方法です。

📘 単元別学習ノート

🔤

英文法 (Grammar)

品詞、文型、時制、受動態、関係詞、仮定法 — 大学受験に必須の文法を完全整理

時制 = 時間の地図 — will/would/have+ppの関係 📝

語彙 (Vocabulary)

頻出単語、派生語、コロケーション、文脈推測 — 語彙力拡張戦略

語根+接辞 — un+happy, re+view どんな単語も推測できる 🔍

読解 (Reading)

主題・要旨把握、空所補充、内容一致 — 共通テスト・二次試験対策

英語読解 = まず設問確認、目的を持って読む ✍️

英作文 (Writing)

和文英訳、自由英作文、要約 — 記述問題完全攻略

英語の語順 = 動詞が先 — 日本語とは逆の思考順序

📚 レベル別学習ガイド

中学英語 (1〜3年)

  • 文法: 基本時制（現在・過去・未来）、be動詞・一般動詞、助動詞
  • 語彙: 基礎語彙1,000〜2,000語、日常生活表現
  • 読解: 短い英文の理解、主題・詳細情報の把握
  • 作文: 基本文・複文作成、メール・日記の形式

高校英語 (1〜3年 / 大学受験対策)

  • 文法: 関係詞、仮定法、分詞構文、倒置・強調構文、共通テスト文法対策
  • 語彙: 大学受験頻出語彙3,000+、多義語、同義語・反義語
  • 読解: 長文読解、空所補充、内容一致、英文要約
  • 作文: 要約文、条件英作文、記述式解答
📌 大学受験英語 対策

共通テスト・二次試験英語の高得点のために: 語彙力 → 構文理解 → 読解速度 の順に実力をつけましょう。教科書・問題集を中心に、様々な形式の問題を解くことが効果的です。
