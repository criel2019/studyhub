🔍 英語読解の戦略

🌐 高校英語 / 大学受験対策 🎯 重要：主題把握·空所補充·文整序·長文
直感的に理解しよう

英語の読解は推測の技術です。知らない単語が出てきても、文脈・文構造・語根から意味を推測できます。設問を先に確認する習慣が重要です。

1. 読解の基本原則

  • 文章構造の把握：序論（主題提示）→ 本論（根拠·例示）→ 結論（まとめ·強調）
  • 主題文を探す：通常は第1文または最終文に。逆接語(but/however)の後に核心内容が多い
  • 選択肢の分析：具体的すぎる·広すぎる選択肢を消去
  • 時間管理：1問あたり平均1分30秒〜2分を目安に

2. 主題·要旨·タイトル把握

解法：
  1. 第1文と最終文を先に読んで文章の方向を把握
  2. 繰り返し登場するキーワードを確認（主題に直結）
  3. 逆接語(However/But/Yet)の後の内容 → 筆者の核心主張
  4. タイトルは主題を含意的·比喩的に表現したもの

3. 空所補充（空欄推論）

大学入試で最も難しく、配点の高いタイプ。空所 = 文章の核心内容。
  1. 空所を含む文の意味·構造を把握
  2. 空所の前後から手がかり（例示·根拠·言い換え）を探す
  3. 文章全体の論理の流れを把握
  4. 各選択肢を空所に当てはめて論理的に合うか確認
重要原則：空所に入る内容は必ず文章内に根拠があります。
自分の背景知識ではなく、本文内の情報だけで判断してください。

4. 文整序·段落整序

  • 文整序：指示語(this/that/it/they)と接続語で前後関係を把握
    最初の文のキーワード → 続く段落で同じキーワードを探す
  • 段落整序：各段落の最初と最後の文に注目して繋がりを確認
  • 接続語の活用：First/Then/Finally（順序）、However（対比）、Therefore（結果）、For example（例示）

5. 内容一致問題

  1. 選択肢を先に読んでキーワードを把握
  2. 本文の該当箇所を探して照合
  3. 本文と選択肢の表現が言い換えられていることが多い（同義語·パラフレーズ）
  4. 部分否定·全体否定の違いに注意（not all ≠ none）

6. 読解速度を上げる方法

  • 句単位で読む：単語ずつではなく、意味のまとまりで区切って読む
  • 主語·述語を素早く把握：修飾語句を省いて主節を先に理解
  • 逆読みを減らす：前から後ろへ一度で読む習慣
  • 多読の習慣：英語の長文に日常的に触れることで読むスピードが上がる
💡 大学受験英語読解のコツ：空所補充·文整序問題は文章の論理構造を把握することが核心です。「なぜこの文がここにあるのか?」と自問する練習を通じて、論理的読解力を養いましょう。
