📝 英語語彙の学習

🌐 中学·高校 英語 🎯 重要戦略：語根·文脈推論·コロケーション·多義語

1. 語彙力とは

語彙力とは単に単語を多く知ることではなく、文脈の中で意味を正確に把握し、使い分ける力です。
  • 受動語彙：読む·聞くときに理解できる語（読解の核心）
  • 能動語彙：自分で書いたり話したりできる語（英作文·記述に必要）
  • 大学受験基準：約5,000~7,000語が目安

2. 語根·接頭語·接尾語

接辞を知れば未知語の意味を推測できます。
  • 接頭語：un-(not: unhappy), dis-(not: disagree), re-(again: review), pre-(before: predict), over-(too much: overwork), mis-(wrong: mislead)
  • 接尾語（名詞）：-tion/-sion(action), -ment(agreement), -ness(happiness), -ity(ability)
  • 接尾語（形容詞）：-ful(useful), -less(careless), -able(possible), -ous(famous), -al(natural)
  • 接尾語（動詞）：-ize(realize), -ify(simplify), -en(widen)
派生語グループ例：
create → creation(名) → creative(形) → creatively(副) → creativity(名)
depend → dependent(形) → independence(名) → independently(副)

3. 文脈からの語義推論

知らない単語が出ても文脈から意味を推測する戦略：
  • 定義型："X, which means / that is / in other words" → 直後に定義
  • 例示型："such as, for example, including" → 例から上位概念を把握
  • 対比型："but, however, although, unlike" → 反対の意味から推測
  • 因果型："because, therefore, as a result" → 原因·結果関係で絞り込み
練習："The new policy was contentious; many opposed it while others strongly supported it."
→ opposed vs supported → 相反する反応 → contentious = 論争的な、賛否が分かれる

4. 入試頻出語彙テーマ別

  • 心理·感情：anxiety(不安), reluctant(気が乗らない), contempt(軽蔑), empathy(共感), nostalgic(懐かしい)
  • 社会·文化：diverse(多様な), prevailing(支配的な), conform(順応する), prejudice(偏見), hierarchy(階層)
  • 科学·技術：phenomenon(現象), hypothesis(仮説), inevitable(不可避の), fluctuate(変動する)
  • 論理·接続：nevertheless(それにもかかわらず), whereas(〜する一方), consequently(結果として)

5. 多義語とコロケーション

  • 多義語：文脈によって意味が異なる語
    "light" → 光·軽い·明るい·点ける、"run" → 走る·運営する·流れる
  • コロケーション：よく一緒に使われる語の組み合わせ
    make a decision(✓) vs do a decision(✗)
    take a risk, pay attention, give advice
  • 類義語の使い分け："big/large/great"、"say/tell/speak/talk" — ニュアンスの違いを意識

6. 語彙拡張戦略

  • 例文·派生語とセットで記録：単語帳に用例と派生語を必ず書く
  • テーマ別グループ学習：関連語をまとめて覚える（例：環境語彙セット）
  • 多読の習慣：英語の教科書·入試問題·英字新聞に日常的に触れる
  • 実際に使ってみる：英作文の中で新しい語を使うと記憶が定着する
💡 英単語学習のコツ：単語は文脈の中で覚えるのが最も効果的です。単語帳の丸暗記より、実際の英文の中で使われる様子を見て理解しましょう。「この単語は何と一緒に使われるか（コロケーション）」を意識すると、使える語彙が増えます。
